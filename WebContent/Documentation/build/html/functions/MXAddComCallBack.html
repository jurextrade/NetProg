292
293      <p><font color="#000080" face="Verdana" size="2">This function adds a
294      callback
295to the specified message class callback list. The callback is triggered when a message
296      belonging to (dialogclassname, messageclassname) is received or sent on
297      the connection instance <i>pcom</i><br>
298      <br>
299      The <i> mode</i> parameter can be : MXONRECV, MXONSEND, MXONRECEIVING, MXONSENDING</font></p>
300
301<p><font face="Verdana" size="2" color="#000080"><i>applicationfield</i> specifies
302the argument that is to be passed to the specified procedure when it is invoked</font></p>
