MXEnd

#include “mx.h”

void MXEnd(MX *pmx)
Parameters

  • pmx – the MX manager

Returns

no return value

Description

This function terminates the MX session of an application. Usually it is called at the end of the program before exiting. It frees all resources, closes all the connections.

See Also

MXInit()

Example
 1 int main ()
 2 {
 3   MX mx;
 4
 5   MXInit (&mx, MXSERVER, NULL, NULL, 0, NULL);
 6   MXAddPort (&mx, 7080, IPPROTO_TCP);
 7   MXAddGeneralConnectCallBack (&mx, MXONCONNECT, TCPOnConnect, NULL);
 8   MXAddGeneralConnectCallBack (&mx, MXONCLOSE, TCPOnClose, NULL);
 9
10   MXDispatchEvents (&mx, 0);
11
12   MXEnd (&mx);
13 }
